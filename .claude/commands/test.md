# /test — run tests locally

Run unit, integration, and/or e2e tests locally:

| Arguments | Scope |
|---|---|
| `unit` | Run all of the unit tests, but only the unit tests |
| `integration` | Run all of the integration tests, but only the integration tests |
| `e2e` | Run all of the e2e tests, but only the e2e tests |
| `all` | Run all of the unit, integration, and e2e tests |

## Rules

- Before running tests, check to see if the services are running locally and on the correct ports
- Try to fix issues as you encounter them
